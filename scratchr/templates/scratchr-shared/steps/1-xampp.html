Downloading

First of all, let's navigate to the download page. The link for it will vary depending on your operating system. Note that this guide expects you to be on Windows.

  • Windows
  • Linux
  • macOS / Mac OS X

All of these files may seem daunting, but in reality, we only need xampp-win32-1.7.7-VC9.zip. Click on it to start the download.

Installing

Now, run it. If you get a warning saying something in the lines of "Do you want to allow this app to make changes to your device?", click Yes. When it asks you to select a language, you can either select a language or keep english and then press OK.

You are now in the installer!

Click next, click it again, and again, then click Install.

XAMPP is now installed!

It will then ask you if you want to start the XAMPP Control Panel. Click Yes, it'll be useful for later.

Now with all of this done, head to Installing Memcached
