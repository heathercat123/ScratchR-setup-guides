Installing git

First of all, let's navigate to the download page. The link for it will vary depending on your operating system. Note that this guide expects you to be on Windows.

  • Windows 10+
  • Windows 7+
  • Linux
  • macOS / Mac OS X

Run the installer to install git

Cloning ScratchR

Navigate to C:\XAMPP\, then delete htdocs.

Next, open up CMD like with memcached, but this time, run this command:


git svn clone https://subversion.assembla.com/svn/scratchr/


In the next steps, if I mention anything under \app, it's inside C:\XAMPP\htdocs\scratchr\app

Now with all of this done, head to Setting up the database
